# frontend

This folder contains the frontend for the campaign budget tracker. Technologies used: Vue, Vuetify.

Please find instructions below on how to run the frontend locally.

Ensure you are in the frontend folder when running the below steps.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
